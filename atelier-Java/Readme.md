## Compile to C

run the following commands :    
                                                                              
```
$ sudo apt-get install flex                               
$ git clone https://github.com/MahamdiAmine/TPGIO-tp1.git                                
$ cd TPGIO-tp1/atelier1/atelier-C/
$ flex tp.lex
$ gcc lex.yy.c -o a.o -lfl
$ ./file.o
```
